# Optimizing-E-KYC-using-distributed-ledger-technology-and-smart-contracts
A blockchain-based e-KYC solution that leverages Distributed Ledger Technology (DLT), Smart Contracts, and IPFS to provide secure, transparent, tamper-resistant, and decentralized identity verification.Blockchain-powered e-KYC platform utilizing Ethereum Smart Contracts and IPFS


## Project Overview

This project introduces a secure Electronic Know Your Customer (E-KYC) system using Blockchain Technology and Smart Contracts. Customer identity information is securely stored using IPFS while transaction records are maintained on the Ethereum blockchain, ensuring transparency, integrity, and tamper-proof verification.

---

##  Features

- Secure User Registration
- Blockchain-based Identity Verification
- Smart Contract Automation
- IPFS File Storage
- Tamper-proof Data
- Secure Authentication
- Ethereum Transaction Logging
